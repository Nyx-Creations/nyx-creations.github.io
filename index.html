Basic Elements

Keywords

Keywords are reserved words that have special meaning in Ouro and cannot be used as identifiers (variable names, function names, etc.).

Keyword Description
if Used for conditional execution.
else Used with if for alternative execution.
while Creates a loop that executes as long as a condition is true.
void Declares a new function.
public Declares a new function.
private Declares a new function.
protected Declares a new function.
class Declares a new function.
enum Declares a new function.
struct Declares a new function.
function Declares a new function.

Identifiers

Identifiers are names used to identify variables, functions, types, etc. In Ouro, identifiers must:

  • Start with a letter (a-z, A-Z) or an underscore (_).
  • Be followed by letters, numbers (0-9), or underscores.
  • Be case-sensitive/case-insensitive (specify which).
  • Not be a reserved keyword.
// Example of valid identifiers
myVariable
_another_one
calculateSum123

// Example of invalid identifiers
123start (starts with a number)
my-variable (contains a hyphen)
if (is a keyword)

Control Flow

Control flow statements dictate the order in which statements are executed.

Conditional Statements (If-Else)

// Basic If
if (condition) {
    // code to execute if condition is true
}

// If-Else
if (condition) {
    // code if true
} else {
    // code if false
}

// If-Else If-Else
if (condition1) {
    // code if condition1 is true
} else if (condition2) {
    // code if condition2 is true
} else {
    // code if no conditions are true
}

Looping Statements (While, For)

While Loop

while (condition) {
    // code to repeat as long as condition is true
}

Functions

Functions are blocks of code designed to perform a particular task.

Function Declaration

// Syntax: [keyword] [functionName]([parameters]) [optional_return_type] {
//     // function body
//     return value; // if applicable
// }

function greet(name: String): String {
    return "Hello, " + name + "!";
}

function calculateSum(a: Integer, b: Integer) {
    print(a + b);
}

Function Call

let message = greet("Alice");
calculateSum(5, 7);

Comments

Comments are non-executable text within the code, used for explanation and documentation.

Single-line Comments

// This is a single-line comment.
let x = 10; // Comment after a statement.

Multi-line Comments

/*
This is a multi-line comment.
It can span across multiple lines.
*/

Examples

A few complete examples demonstrating various syntax features.

Example 1: Basic Program

// This is a simple program in Ouro

function main() {
    print("Hello from Ouro!");
    let count = 0;
    while (count < 3) {
        print("Count: " + count);
        count = count + 1;
    }
}

main(); // Call the main function

Example 2: Function with Conditional Logic

function checkNumber(num: Integer): String {
    if (num > 0) {
        return "Positive";
    } else if (num < 0) {
        return "Negative";
    } else {
        return "Zero";
    }
}

print(checkNumber(10));  // Output: Positive
print(checkNumber(-5));  // Output: Negative
print(checkNumber(0));   // Output: Zero
